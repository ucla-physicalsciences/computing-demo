Web Hosting & Development


The Computing Group provides web development, web hosting, and content management services for research and administrative groups. Research groups may request a website to be set up for them.

Web Development

Research groups may request website which is self-managed or managed by the computing group using open-source Bootstrap themes. The website will be hosted on the department's highly available web cluster.

For security reasons we do not offer Drupal or Wordpress platforms within the Physics & Astronomy network.

Web Hosting

The department's websites are hosted on a load-balanced highly-available web cluster.

Content Management

The computing group provides assistance with content maintenance of the department's websites upon request.

For questions about web development, web hosting, or content management, please submit a help desk request .
