Connecting To The PnA VPN


Intructions for connecting to the Physics & Astronomy VPN are available for Mac and PC. Note that you will need to sign in with your Physics & Astronomy department e-mail username and password to view these articles.

Please select your platform:

Windows XP, Vista, 7, or 8

To connect to the UCLA PnA VPN from a Windows XP, Vista, 7, or 8 computer:

  1. Download and install the Cisco VPN Client software
    1. Dowload the Cisco VPN Client software (32-bit, Windows XP, Vista, 7, 8)
    2. Download the Cisco VPN Client software (64-bix, Windows XP, Vista, 7, 8)
  2. Download the UCLA PnA VPN Profile (right click, save-as)
  3. Run the VPN Client program
  4. Click Import
  5. Select the UCLA PnA VPN Profile
  6. Click Connect
  7. Enter your Physics & Astronomy department e-mail username and password
Note for Windows 8

In Windows 8 an error will appear when attempting to connect to the VPN with the message "Failed to enable virtual adapter." To resolve this issue:

  1. Open Registry editor by typing regedit in the Run prompt
  2. Browse to the Registry Key HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\CVirtA
  3. Select the DisplayName to modify, and remove the leading characters from the value data upto "%;" i.e.
    1. For x86, change the value data from something like "@oem8.inf,%CVirtA_Desc%;Cisco Systems VPN Adapter” to "Cisco Systems VPN Adapter”
    2. For x64, change the value data from something like "@oem8.inf,%CVirtA_Desc%;Cisco Systems VPN Adapter for 64-bit Windows” to "Cisco Systems VPN Adapter for 64-bit Windows”
  4. Try connecting again
Note for Windows 8.1

The Cisco VPN client software is not supported on Windows 8.1. The department is in the process of deploying replacement VPN endpoints that will allow for connectivity from Windows 8.1. However, tHERE is no specific ETA at the moment. THERE are no recommended workarounds in the meantime.

To connect to the UCLA Physics & Astronomy VPN from various platforms: Windows 7/8/10, Mac OS, iOS, Android, Chrome OS (Chromebooks).

  1. Download the appropriate Cisco AnyConnect client installer from

    https://www.it.ucla.edu/it-support-center/services/virtual-private-netwo...

    (You will need to use your BOL credentials to access the page.  Download links to our repository will be added later).

  2. Then launch the AnyConnect client app and use the VPN server address, sslvpn.physics.ucla.edu and login using your Physics & Astronomy department e-mail credentials.
