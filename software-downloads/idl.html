Download IDL


The department has purchased a volume license for IDL that can be used for internal academic purposes. Licenses are temporarily reserved from a central licensing server on a "floating" basis, so for IDL to function the workstation must be connected to the PnA network by wired connection, PA Internal wireless network, or VPN .

Installation files and instructions are provided below. If you encounter difficulty installing IDL on a workstation, please submit a new help desk request.

Installation Files (latest version)

IDL 8.7 (64-bit)

  • Mac OS X Native
  • Mac OS X Unix (remove the _ from .tar_.)
  • Windows (7,8,10)
  • Linux
Installation Instructions (latest version)
  1. Download the installation file for your operating system above
  2. Install IDL, and when it asks for the license server, point it to:
    Host: idl.pa.ucla.edu
    Port: 7070
  3. If it does not ask you automatically, please run the 'harrislicense' or 'harrislicense87' to set the license server.
